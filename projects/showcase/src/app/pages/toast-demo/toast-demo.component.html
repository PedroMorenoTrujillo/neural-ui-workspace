NeuToastService — Métodos

Método {{ 'demo.common.api.type' | transloco }} {{ 'demo.common.api.desc' | transloco }}
show(options) string Muestra un toast con opciones completas. Devuelve el id.
success(msg, opts?) string Atajo para tipo success. Devuelve el id.
error(msg, opts?) string Atajo para tipo error.
warning(msg, opts?) string Atajo para tipo warning.
info(msg, opts?) string Atajo para tipo info.
dismiss(id) void Cierra el toast con el id dado.
clear() void Cierra todos los toasts activos.
setPosition(pos) void Cambia la posición del contenedor. La posición se mantiene hasta cambiarse de nuevo.

NeuToastOptions

Propiedad {{ 'demo.common.api.type' | transloco }} Default {{ 'demo.common.api.desc' | transloco }}
message * string — Cuerpo del toast. Requerido.
type 'success'|'error'|'warning'|'info' 'info' Tipo semántico que controla el color y el icono.
title string — Título opcional en negrita sobre el mensaje.
duration number 4000 ms antes de auto-ocultarse. 0 = sin auto-hide (persistente).

NeuToastPosition

Valor {{ 'demo.common.api.desc' | transloco }}
'top-right' Esquina superior derecha (por defecto).
'top-left' Esquina superior izquierda.
'bottom-right' Esquina inferior derecha.
'bottom-left' Esquina inferior izquierda.
