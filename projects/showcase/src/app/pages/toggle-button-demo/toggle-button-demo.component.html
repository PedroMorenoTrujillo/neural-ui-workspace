Componente

Toggle Button Group

Grupo de botones de selección única o múltiple con soporte completo para Angular Forms. Equivalente superior al SelectButton de PrimeNG.

Alineación (con iconos)

Vista — múltiple

Prioridad — texto

Días activos — múltiple, sm
Configurar
Input / Output Tipo Default Descripción
options NeuToggleOption[] [] Lista de opciones. Cada una tiene label, value, icon?, disabled?
multiple boolean false Permite seleccionar varias opciones simultáneamente. El valor del control pasa a ser T[]
size 'sm' | 'md' | 'lg' 'md' Tamaño visual del grupo
disabled boolean false Deshabilita todo el grupo. También soporta setDisabledState de Angular Forms
neuChange EventEmitter<T | T[] | null> — Emite el nuevo valor al seleccionar o deseleccionar
NeuToggleOption.icon string — Nombre de icono Lucide. Requiere registrar el icono en provideIcons()
