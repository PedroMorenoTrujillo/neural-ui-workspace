Interfaz NeuNavItem

Propiedad {{ 'demo.common.api.type' | transloco }} Requisito {{ 'demo.common.api.desc' | transloco }}
id string Requerido Identificador único del ítem
label string Requerido Texto visible del ítem y tooltip en modo colapsado
icon string Requerido Nombre del icono Lucide registrado (ej. lucideHome)
route string Opcional Ruta Angular interna (RouterLink). Excluye href.
href string Opcional URL externa. Se abre en nueva pestaña con rel="noopener noreferrer". Excluye route.
children NeuNavItem[] Opcional Subítems que se renderizan como acordeón. Máximo 3 niveles en total (raíz → hijos → nietos). Los nietos no pueden tener children.
badge string Opcional Texto del badge que aparece a la derecha del label
badgeVariant 'default'|'success'|'warning'|'danger'|'info' Opcional Variante de color del badge ('default' por defecto)
disabled boolean Opcional Desactiva el ítem visualmente e impide la interacción

Inputs

Input {{ 'demo.common.api.type' | transloco }} Default {{ 'demo.common.api.desc' | transloco }}
items NeuNavItem[] [] Array de ítems de navegación
collapsed boolean false Estado inicial colapsado (64 px de ancho)
collapsible boolean true Muestra el botón de colapsar/expandir en la cabecera
ariaLabel string 'Navegación principal' Etiqueta accesible del elemento <nav>

Outputs

Output {{ 'demo.common.api.type' | transloco }} {{ 'demo.common.api.desc' | transloco }}
collapsedChange EventEmitter<boolean> Emite el nuevo estado colapsado cada vez que el usuario hace clic en el botón

Slots (ng-content)

Selector {{ 'demo.common.api.desc' | transloco }}
[neu-nav-brand-icon] Icono/logo compacto mostrado en modo colapsado (64 px). Usa el mismo elemento que [neu-nav-brand] pero solo el icono.
[neu-nav-brand] Zona superior del nav. Acepta logo + nombre de aplicación. Se oculta en modo colapsado.
[neu-nav-footer] Zona inferior del nav. Ideal para datos de usuario o versión. Se oculta en modo colapsado.
