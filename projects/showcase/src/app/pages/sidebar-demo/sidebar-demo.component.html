Componente

Sidebar

Panel lateral cuyo estado abierto/cerrado vive en la URL (?menu=open). Modo overlay en mobile, persistent en desktop ≥400px. El estado es compartible, navegable y sincronizado vía NeuUrlStateService.

Este showcase usa NeuSidebar como navegación principal.
En escritorio (≥400px) es persistente; en mobile es overlay.

URL param: ?menu=open

Patrón URL State NeuUrlStateService.setParam('menu','open')
La URL cambia a ?menu=open y el sidebar se abre reactivamente en cualquier componente que esté escuchando ese parámetro.
Input Tipo Default Descripción
side 'left' | 'right' 'left' Posición del panel: izquierda o derecha de la pantalla
urlParam string 'menu' Nombre del query param URL que controla el estado
persistent boolean false Modo incrustado: sidebar siempre visible integrado en el layout
ariaLabel string 'Menú de navegación' Etiqueta accesible para el elemento <aside>
closeLabel string 'Cerrar menú...' Etiqueta accesible para el botón de cerrar (solo en modo overlay)
