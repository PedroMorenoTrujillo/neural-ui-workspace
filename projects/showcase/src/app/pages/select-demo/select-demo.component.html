Directivas de contenido

Proyecta ng-template dentro de <neu-select> para personalizar el renderizado de cada ítem de la lista o del trigger.

Directiva Selector Contexto implícito {{ 'demo.common.api.desc' | transloco }}
NeuSelectItemDirective [neuSelectItem] NeuSelectOption Template para cada opción en el panel desplegable. Usa let-item para acceder a item.label e item.value.
NeuSelectSelectedDirective [neuSelectSelected] NeuSelectOption | null Template para el trigger cuando hay una opción seleccionada. Si no se provee, muestra el label de la opción.
