Componente

Select

Dropdown personalizado con floating label, teclado y soporte completo para Angular Forms. Cierre automático al hacer clic fuera.

Configurar
Input Tipo Default Descripción
options NeuSelectOption[] [] Lista de opciones
label string '' Floating label
placeholder string 'Seleccionar...' Texto cuando no hay selección
errorMessage string '' Activa estado de error con mensaje
disabled boolean false Deshabilita el select
